    ```typescript
      src,
      width,
      quality,
    }: {
      src: string
      width: number
      quality?: number
    }) {
      return `${src}?w=${width}&q=${quality ?? 75}`
    }
    ```

    **app/globals.css** — Replace entire file with Tailwind v4 CSS-first design system:
    ```css
    @import "tailwindcss";

    @theme {
      /* Brand */
      --color-nvidia: #76b900;

      /* GitHub dark canvas */
      --color-canvas-default: #0d1117;
      --color-canvas-subtle: #161b22;
      --color-canvas-inset: #010409;

      /* Borders */
      --color-border-default: #30363d;
      --color-border-muted: #21262d;

      /* Foreground */
      --color-fg-default: #e6edf3;
      --color-fg-muted: #8b949e;
      --color-fg-subtle: #6e7681;

      /* Responsive breakpoints (mobile-first) */
      --breakpoint-xs: 375px;
      --breakpoint-sm: 640px;
      --breakpoint-md: 768px;
      --breakpoint-lg: 1024px;
      --breakpoint-xl: 1280px;
      --breakpoint-2xl: 1440px;
    }

    /* WCAG enforcement: dark text on NVIDIA green — NEVER text-white on green backgrounds */
    /* contrast ratio: #76b900 on #030712 ≈ 6.5:1 (WCAG AA + AAA pass) */
    .bg-nvidia,
    [class*="bg-nvidia"] {
      color: #030712; /* gray-950 */
    }

    /* Base reset */
    html,
    body {
      background-color: #0d1117;
      color: #e6edf3;
      min-height: 100dvh;
    }

    /* Prevent horizontal overflow on all viewports */
    body {
      overflow-x: hidden;
    }
    ```

    IMPORTANT: Do not create tailwind.config.js or tailwind.config.ts — Tailwind v4 uses CSS-only config via @theme. If shadcn init created a tailwind.config.ts, delete it.

    After writing files, run a build to verify no config errors:

    ```
    pnpm build
    ```
